# Quiz-App-JS
◆ quiz application using HTML,CSS & JavaScript

◈ Technologies Used: HTML, CSS, JavaScript

◈ Description: Developed an interactive quiz application with multiple-choice questions and real-time user interaction.

◈ Functionality: Displays questions dynamically, validates answers, and calculates the final score instantly.

◈ User Interface: Clean and responsive design for a smooth user experience across devices.

◈ Key Concepts: DOM manipulation, event handling, and basic JavaScript logic.
